# Pomodoro Timer

Single-screen React Pomodoro timer with:

- focus and break cycle switching
- start, pause, resume, and reset controls
- `mm:ss` countdown updates every second
- daily history for completed focus sessions
- localStorage persistence across refresh for the same calendar day

## How To Run Locally

1. Install dependencies.

	powershell
	npm install
	

2. Start the development server.

	powershell
	npm run dev
	

3. Open the local URL printed by Vite.

## Build

powershell
npm run build
